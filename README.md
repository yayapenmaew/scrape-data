# scrape-data
a part of ICE senior project used to scrape data and stored in db

Prerequisites:
In terminal (initial terminal, outside the project)
- pip3 install awscli
- aws configure 
  (region = ap-northeast-1, output = json)
In project terminal
- npm install aws-sdk
- npm install uuid
- npm install google-play-scraper

Run:
- node filename.js
